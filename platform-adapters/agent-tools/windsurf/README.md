# Windsurf Agent Tool Adapter

This adapter provides the context file template for [Windsurf](https://windsurf.com/), the AI-first code editor originally built by Codeium. Windsurf reads `.windsurfrules` at the repository root, giving Cascade — Windsurf's autonomous AI agent — immediate access to your project's purpose, architecture, coding conventions, and workflow expectations without any manual setup; copy the `.windsurfrules` template from this directory to your repository root, fill in the placeholders, and Cascade will apply those rules to all AI suggestions and agent interactions from the first session forward. **Research note:** The filename was verified against Windsurf community documentation and the official Codeium blog — `.windsurfrules` is the established convention for the repository root and continues to be supported; more recent Windsurf versions also introduced a directory-based system at `.windsurf/rules/` where rules are defined as named markdown files with activation modes (always on, manual, glob, model decision) — confirm the current recommendation in the [official Windsurf documentation](https://docs.windsurf.com/) before deploying, as Windsurf's tooling evolves frequently.
