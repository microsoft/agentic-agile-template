# Cursor Agent Tool Adapter

This adapter provides the context file template for [Cursor](https://www.cursor.com/), the AI-first code editor. Cursor reads `.cursorrules` at the repository root on every session start, giving the AI assistant immediate access to your project's purpose, architecture, coding conventions, and workflow expectations without any manual setup — copy the `.cursorrules` template from this directory to your repository root, fill in the placeholders, and Cursor will apply those rules to all AI suggestions and agent interactions from the first session forward. **Research note:** The official filename was verified against [https://docs.cursor.com/context/rules](https://docs.cursor.com/context/rules); `.cursorrules` is the widely-adopted legacy convention that Cursor continues to support, while the modern system also allows rule files under `.cursor/rules/` — confirm the current recommendation in the official documentation before deploying, as Cursor's tooling evolves frequently.
