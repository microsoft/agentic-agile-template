# Aider Agent Tool Adapter

This adapter provides the context file template for [aider](https://aider.chat), a command-line AI pair programming tool that lets you edit code in your local git repository using large language models. Aider reads `CONVENTIONS.md` at the repository root when loaded with `aider --read CONVENTIONS.md` or configured via `.aider.conf.yml`, giving the AI immediate access to your project's purpose, architecture, coding conventions, workflow expectations, and boundaries without any manual setup — copy the `CONVENTIONS.md` template from this directory to your repository root, fill in the placeholders, and aider will apply those conventions to all code edits and suggestions in the session. **Research note:** The official conventions filename was verified against [https://aider.chat/docs/usage/conventions.html](https://aider.chat/docs/usage/conventions.html) — `CONVENTIONS.md` is the filename shown in the official aider documentation and example transcripts; confirm the current recommendation in the official documentation before deploying, as aider's tooling evolves frequently.
