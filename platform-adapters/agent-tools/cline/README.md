# Cline Agent Tool Adapter

This adapter provides the context file template for [Cline](https://github.com/cline/cline), an autonomous AI coding agent that runs as a VS Code extension. Cline reads `.clinerules` at the repository root on every session start, giving the agent immediate access to your project's purpose, architecture, coding conventions, workflow expectations, and autonomy boundaries without any manual setup — copy the `.clinerules` template from this directory to your repository root, fill in the placeholders, and Cline will apply those rules to all autonomous operations and interactions in that session. **Research note:** The official filename was verified against [https://github.com/cline/cline](https://github.com/cline/cline) and community documentation — `.clinerules` is the established convention for providing project-specific rules to Cline at the repository root; confirm the current recommendation in the official documentation before deploying, as Cline's tooling evolves frequently.
